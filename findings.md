# Findings
## Research, Discoveries, and Constraints

### Discovery Phase Answers:
**North Star:** 
Generar soluciones de negocio viables para LATAM mediante el análisis automático de videos reales (Starter Story), extrayendo mecánicas de negocio y cruzándolas con problemas locales y perfiles de usuario.

**Integraciones:**
- Apify (YouTube Scraping)
- Supabase/PostgreSQL (Base de datos transaccional central)
- OpenAI API (Análisis y generación)
- Vercel (Next.js Application Hosting)
- Modal (Procesamiento pesado y transcripciones)

**Source of Truth:**
- Canal de YouTube "Starter Story" (Metadata + Transcripciones).

**Delivery Payload:**
- Aplicación Full-stack en Next.js (desplegada en Vercel) mostrando: Videos Procesados, Clasificación de Pain Points en LATAM y Soluciones Personalizadas usando perfil RPM.
